Wallable is a social networking WP plugin. Is is a full features with private wall, dashboard, connections, with all what you want.
It included:
1) Back-end: Manage profiles, comments, email teplates, dashboard icons, etc..
2) Front-end: All features like Facebook or Twitter. It just an shortcode [wallable_main], create page to get URL and save to config from back-end.
All other URLs will render by just one menu item
Note: It is working with hight STYLE with WP base config. All just query to this one shortcode [wallable_main]

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress 
3. Check shortcodes to create views and use it
